# speedrunby

> Personal portfolio of **Amankeldi Kydyraliev** — Claude Certified Architect & fullstack engineer.
> Live: _coming soon at `speedrunby.pages.dev`_

Bilingual (RU default / EN), dark-first theming, a lazy WebGL hero and a hard
performance budget — built as a portfolio artifact in itself: the repository is
meant to be read.

## Stack

Astro 6 (static) · Tailwind CSS v4 · GSAP + Lenis · three.js (single lazy island) · TypeScript strict · pnpm

## Getting started

```sh
pnpm install
pnpm dev       # local dev server
pnpm build     # production build to dist/
pnpm preview   # serve the production build
```

Requires Node ≥ 22 (`.node-version`) and pnpm 10 (`packageManager`).
