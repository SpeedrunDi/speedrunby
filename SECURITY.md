# Security Policy

This is a static personal portfolio site with no user data, no authentication
and no server-side runtime.

If you believe you've found a security issue (e.g. in the deployed headers,
the optional contact worker, or a dependency), please report it privately:

- **Email:** amanmaksat0@gmail.com

Please do not open public issues for security reports. There is no bug bounty,
but reports are appreciated and will be acknowledged.
